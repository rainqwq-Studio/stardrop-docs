# 一、服务器守则

1. 请遵守中华人民共和国与您所在的地区的法律法规，并对自己的行为所产生的任何后果负责

2. 文明交流，用语得体，切勿诽谤、辱骂他人，不要刷屏

3. 严禁在社区内任何地方发布涉政、暴力、低俗、恐怖等不良内容，禁止煽动对立

4. 文明游玩，切勿破坏他人的游玩成果。严禁攻击、压力测试服务器及其附属设施。建设大型工程建议在群内说明与报备

5. 服从管理员的决定，如有争议请及时告知

6. 注意玩梗程度，请勿乱玩梗、玩烂梗

7. AIGC内容（AI生成内容）请确保他人能够辨认出，如无明显特征，请另外注明为AIGC内容
